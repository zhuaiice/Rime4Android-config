# Rime-android
安卓端Rime自用配置

**sogou主题布局符号表情输出更友好建议使用**
# 特点
1. 小鹤双拼
2. 中英
3. gboard墨色配色
# 用法
下载-覆盖-部署

建议克隆或download zip，不建议到release下
